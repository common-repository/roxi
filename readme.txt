This plugin allows the Roxi widget to be quickly and easily inserted into any page or post. Requires a venue account at getroxi.com.

== Description ==

Usage of this plugin requires a [Roxi](https://getroxi.com) venue account.

This plugin will allow you to quickly and easily insert the Roxi widget into any page or post in your blog.

With the widget, visitors to your website will be able to:

* View events
* Get on guestlist
* Book bottle service
* Purchase admission tickets

All of which can be done without leaving your website.

The plugin also allows you to configure the look and feel of the widget, choosing between different styles and themes to better match your blog.

== Installation ==

1. Upload the `roxi` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your Roxi Venue ID in the 'Settings' page of the plugin.
4. Create a new post or page, or edit an existing one, and use the Roxi button to insert the widget anywhere you choose.

== Frequently Asked Questions ==

= How can I set my venue ID, or change the style of the widget? =

You can configure these options through the plugin's settings page. The settings page can be found in the left menu of the WordPress admin, under settings. It is titled 'Roxi Settings'.
